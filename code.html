Python script for downloading and processing GOES-R data

Written by Danielle Losos using Google Colab and Jupyter Notebooks.

The following Python code was created to process a sample of GOES-R surface products, including a calculation of the Normalized Difference Vegetation Index (NDVI) and the Near Infrared Reflectance of Vegetation (NIRv) from surface reflectances, as well as an estimate of Photosynthetically Active Radiation (PAR) and NIRv from the Downward Shortwave Radiation (DSR). The data repositories required for processing can be found through the Ameriflux network and the gcp-public-data-goes-16 bucket available on Google Cloud.

GOES_ABI_LandSurfaceProduct_Downloads.ipynb

ABI_LandSurfaceProducts_Download samples several GOES-R surface data products (including Surface Reflection, Temperature, and Downward Shortwave Radiation) at point locations, effectively converting stacks of images into time-series stored in csv files. This code loops through each product at each point location at every timestamp in a given year.
